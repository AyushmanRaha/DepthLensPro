    ◈ About DepthLens Pro

    DepthLens Pro is a full-stack machine vision application for monocular depth estimation, built as a course project for Machine Vision. It transforms 2D images into detailed depth maps using the MiDaS family of neural networks.

    Architecture

    Frontend

    Vanilla HTML/CSS/JS with Chart.js. Responsive dark UI with CSS animations and drag-and-drop file handling.

    Backend

    FastAPI (Python) with async request handling, input validation, result caching, and structured JSON responses.

    ML Models

    PyTorch MiDaS via Torch Hub. Supports MiDaS Small, DPT-Hybrid, and DPT-Large with 8 colourmap options.

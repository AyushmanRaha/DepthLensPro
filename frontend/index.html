    ▣ About DepthLens Pro

    DepthLens Pro is a full-stack machine vision application for monocular depth estimation. Built for the Machine Vision course, it transforms 2D images into 3D depth maps using the MiDaS neural network family.

    Architecture

    Frontend

    Vanilla HTML/CSS/JS + Chart.js. Responsive dark UI, drag-and-drop, lightbox, and persistent settings via localStorage.

    Backend

    FastAPI (Python) with device-aware model loading, in-process caching, and 20+ MDE metrics computed per inference.

    ML Models

    PyTorch MiDaS via Torch Hub. Three models with different speed/accuracy trade-offs. 8 colourmap options.

    Running Locally

    # 1. Start the inference engine (FastAPI) python -m uvicorn backend.app:app --host 127.0.0.1 --port 8765 # 2. Serve the frontend cd frontend python3 -m http.server 3000 # Electron uses http://127.0.0.1:8765 by default

    About the Metrics

    Metrics computed per depth map are split into two categories:

    Computable (no ground truth needed) — SSIM, gradient error, edge density, PSNR, entropy, SILog, and distribution stats are derived directly from the predicted depth map and the input image.

    GT-required — Abs Rel, Sq Rel, threshold accuracy (δ<1.25ⁿ), LPIPS, and ordinal error require a reference depth map (e.g. from LiDAR or stereo). These show "N/A" since only the source image is available.
